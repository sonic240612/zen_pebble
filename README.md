# 🗿 The Zen Pebble

> **배포 링크:** [zen-pebble.vercel.app](https://zen-pebble.vercel.app)

마우스와 키보드에서 손을 떼고 가장 오래 버티는 사람이 승리하는 **디지털 인내심 테스트**.

시간이 지날수록 조약돌 주변의 환경이 변화합니다. 당신은 얼마나 버틸 수 있나요?

## 기능

- 8단계 환경 진화 (새벽 → 아침 → 낮 → 오후 → 황혼 → 밤 → 깊은 밤 → 새벽 복귀)
- 7,680+ 환경 조합 (풍경 × 계절 × 날씨 × 동물 × 천문 현상)
- 48시간 윤회 시스템
- 모바일 대응 (터치/자이로 센서)
- 3D 조약돌 렌더링

## 기술 스택

- **Frontend:** React + TypeScript + Three.js (React Three Fiber)
- **State:** Zustand
- **Build:** Vite
- **Deploy:** Vercel

## 실행

```bash
npm install
npm run dev
```
